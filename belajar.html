GIT HUB

Cara menginstal dan menguapload sebuah file di Git Hub.

Pengertian Git Hub.

GitHub adalah platform pengembangan perangkat lunak yang memfasilitasi kolaborasi, pengelolaan proyek, dan pengunggahan kode sumber. Ini adalah tempat di mana pengembang perangkat lunak, tim pengembang, dan komunitas dapat bekerja bersama untuk membuat, menguji, dan memelihara perangkat lunak.

Langkah-langkah Menginstal Git Hub.

  1. Unduh instalator Git
    • Buka situs web resmi Git: https://git-scm.com/.
    • Di halaman beranda, Anda akan melihat tombol "Download for Windows". Klik tombol ini untuk mengunduh instalator Git untuk Windows.
  2. Jalankan Instalator
    • Setelah unduhan selesai, buka file instalator Git yang telah Anda unduh.
    • Jika muncul infomasi lisensi Git, klik Next > untuk melanjutkan.
    • Selanjutnya menentukan lokasi instalasi. Biarkan saja apa adanya, kemudian klik Next >.
    • Dalam pemilihan komoponen, biarkan saja dan lanjut klik Next >.
    • Selanjutnya pemlilihan direktori start menu, klik Next >.
    • Selanjutnya pengaturan PATH Environment. Pilih yang tengah agar perintah git dapat di kenali di Command Prompt (CMD). Setelah itu klik Next >.
    • Selanjutnya konversi line ending. Biarkan saja seperti ini, kemudian klik Next >.
    • Selanjutnya pemilihan emulator terminal. Pilih saja yang bawah, kemudian klik Next >.
    • Selanjutnya pemilihan opsi ekstra. Klik saja Next >.
    • Selanjutnya pemilihan opsi ekspreimental, langsung saja klik Install untuk memaulai instalasi.
    • Tunggu beberapa saat, instalasi sedang dilakukan. Ketika selesai maka klik finish
    • Selamat, Git sudah terinstal di Windows. Untuk mencobanya, silahkan buka CMD atau PowerShell, kemudian ketik perintah git --version.

Konfigurasi Git Hub.

Ada beberapa konfigurasi yang harus dupersiapakan sebelum mulai menggunakan Git, seperti name dan email. Silahkan lakukan konfigurasi dengan perintah berikut ini;

  • git config --global user.name "Petani Kode"
  • git config --global user.email contoh@petanikode.com

Kemudian periksa konfigurasinya dengan perintah;

  • git config --list

Apabila berhasil maka akan muncul seperti dibawah. Jika sudah muncul seperti itu maka konfigurasi anda berhasil.

  • user.email=Apresgode21
  • user.name=Apres
Langgkah-langkah Upload File di Git Hub.

  1. Buat Repository Baru Setelah masuk ke akun Github, kamu harus membuat repository baru untuk projectmu. Klik tombol “New repository” dan isi informasi yang diperlukan seperti nama repository, deskripsi, dan pilihan privasi. Pilih opsi “Public” jika kamu ingin membagikan projectmu ke dunia, atau “Private” jika hanya ingin membagikannya ke tim tertentu.
  2. Upload Project ke Repository Setelah membuat repository baru, kamu dapat mulai mengupload projectmu. Ada dua cara untuk melakukan ini:
    • menggunakan fitur drag and drop atau menggunakan Git. Untuk metode drag and drop, kamu hanya perlu mengarahkan file-file projectmu ke area drag and drop pada halaman repository.
    • Untuk metode Git, kamu perlu menginstall Git terlebih dahulu dan mengikuti langkah-langkah berikut:
      1. Buka terminal atau command prompt
      2. Masuk ke folder projectmu dengan perintah cd [nama folder]
      3. Inisialisasi Git di folder tersebut dengan perintah git init
      4. Tambahkan file-file projectmu ke repository dengan perintah git add [nama file]
      5. Commit perubahan dengan perintah git commit -m [pesan commit]
      6. Tambahkan URL repository Github sebagai remote repository dengan perintah git remote add origin [url repository]
      7. Upload projectmu ke repository dengan perintah git push -u origin master
  3. Lihat Hasil Upload Project Setelah kamu selesai mengupload projectmu, kamu dapat melihat hasilnya di halaman repository Github. Semua file dan folder yang kamu upload akan ditampilkan di sana, dan kamu juga dapat melihat histori perubahan dan mengelola file dengan fitur Github seperti pull request, branch, dan lain-lain.
